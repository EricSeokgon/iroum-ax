## 2. 제품 비전: Anchor → Platform 전략

### 2.1 1차 제품 (Anchor)

**1차 도메인**: 공기업 경영평가 보고서 자동화 AI 솔루션

**Anchor 고객**: 한국전력기술 (KEPCO E&C)

**1차 제공 가치**:
- 경영평가팀이 1년 내내 수행하던 실적보고서 작성·검토·개선 업무를 **수주(數週) 단위로 단축**
- 평가기준(안전보건) 대비 자사 역량 갭(Gap) 자동 분석
- 등급 상승 시나리오 자동 시뮬레이션 (B 등급 → A 등급 가능성 평가)

### 2.2 2차 확장 (Platform)

**2년차 로드맵**: 동일 아키텍처로 공공기관 규제 도메인 수직 확장

| 연차 | 도메인 | 대상 기관 | 사용자 수 |
|------|--------|---------|----------|
| 1 | 경영평가 (안전보건) | KEPCO E&C | 5-10명 |
| 2 | ESG 보고서 | 대형 공기업 | 30-50명 |
| 2 | 감사 보고서 | 준정부기관 | 20-30명 |
| 2 | 면허신청서 | 중소 공기업 | 10-20명 |
| 3 | 정부 공문 자동화 | 전 공공기관 | 300+ 명 |

**플랫폼 비전**: 한국 공공기관 기록(문서)·규제(평가) 자동화의 표준 인프라

## 3. Anchor Use Case: 공기업 경영평가 PoC

### 3.1 사용자 통점(Pain Point)

| 구분 | 현황 | 개선 목표 |
|------|------|----------|
| **인력·업무량** | 경영평가팀 5-10명 × 연간 풀타임 | 단축 사이클로 전환 (단축 비율은 PoC 결과로 측정) |
| **보고서 품질** | 담당자 개인 경험에 의존, 등급 차이 발생 가능 | 평가기준 준수 자동화, 등급 보정 가능 |
| **의사결정** | B등급 고착, 상향 근거 불명확 | A등급 도달 가능성/콘텐츠 시뮬레이션 제공 |
| **기관 가치** | 경영평가 등급 = 기관장 성과급, 기관 명예 | 등급 1단계 상향 = 기관 예산·명성 향상 |

### 3.2 PoC 기본 정보

**PoC 평가항목**: "안전 보건" (경영평가 500개 항목 중 1개 선정)

**PoC 단계**: Phase 1 — Anchor 고객 확보 (우선순위 High)

**고객 제공 자료**:
1. 기획재정부 경영평가 편람 (안전보건 평가기준 및 배점)
2. 경영평가 실적보고서 작성지침 (공문 작성 규칙)
3. 한국전력기술 자사 실적보고서 (작성 인스턴스)
4. A/B/C/D 등급 타 기관 안전보건 실적보고서 (등급별 벤치마크 학습용)

**고객이 AI 에이전트로 기대하는 기능**:
1. **Document Ingestion** — HWP/PDF 자료 파싱 → 평가기준 & 고객 콘텐츠 자동 추출
2. **기준 매핑** — 경영평가 편람 + 작성지침 → RAG 벡터 인덱싱
3. **등급 시뮬레이션** — 타 기관 A/B/C/D 보고서 학습 기반 자사 예상등급 계산
4. **보고서 초안 생성** — 평가기준 준수 + 한국어 공문 스타일 보고서 섹션 자동 작성
5. **Recommendation 엔진** — 현재 등급(B) 대비 상향(A) 달성에 필요한 콘텐츠 항목 제안

### 3.3 기술 차별화 요소

| 차별화 요소 | 기존 솔루션(LangGraph/CrewAI/Copilot Studio) | iroum-ax |
|-----------|-----------------------------------------------|-----------|
| **언어** | 영어 기반 (한국어 지원 미흡) | 한국어 우선 설계 |
| **도메인** | 범용 Agentic 플랫폼 | 한국 공공 규제/보고 도메인 특화 |
| **문서 형식** | PDF/DOCX 기본 지원 | **HWP(한글) 네이티브 파싱** |
| **배포** | 클라우드/SaaS 기본 | **K8s 네이티브 + 셀프호스팅** (망분리 정합) |
| **LLM 선택** | API 의존 (OpenAI/Claude) | **자체 호스팅 오픈소스 LLM** (EXAONE/Qwen, 데이터 주권) |
| **시장 포지션** | 수평적 인프라 | 수직적 도메인 솔루션 |

## 5. MVP 핵심 기능 (5가지)

### 5.1 Document Ingestion (문서 수집·파싱)

**입력**: HWP/PDF 파일, 스캔 이미지

**처리**:
- VLM 기반 OCR (Qwen2-VL)
- 테이블·그래프 인식
- 페이지 구조 분석

**출력**: 구조화된 텍스트 + 메타데이터 (작성자, 작성일, 부서)

**성공 지표**: OCR 정확도 95%+, 처리 속도 페이지/초 2+

---

### 5.2 기준 매핑 (Criterion Mapping & RAG Indexing)

**입력**: 
- 경영평가 편람 (평가기준 정의)
- 작성지침 (문서 작성 규칙)
- 고객사 실적보고서 (콘텐츠 샘플)

**처리**:
- 평가기준 구조 분석 (계층: 항목 → 지표 → 배점)
- 한국어 임베딩 모델 기반 RAG 인덱싱
- 기준 ↔ 콘텐츠 관계 매핑

**출력**: 검색 가능한 벡터 DB (평가기준별 관련 콘텐츠)

**성공 지표**: 평가기준 검색 정확도 90%+, 관련성 스코어 0.8+

---

### 5.3 등급 시뮬레이션 (Grade Simulation Engine)

**입력**:
- A/B/C/D 등급의 타 기관 실적보고서 (벤치마크)
- 자사 초안 보고서

**처리**:
- 등급별 보고서 학습 (핵심 콘텐츠 특징 추출)
- 자사 보고서 대비 적격성 점수 계산
- 등급별 확률 시뮬레이션 (50% B, 30% A, 20% C 등)

**출력**: 예상등급, 상향 필요 콘텐츠 항목, 시나리오별 점수

**성공 지표**: 등급 예측 정확도 80%+, 상향 시나리오 3-5개 제안

---

### 5.4 보고서 초안 생성 (Report Draft Generation)

**입력**:
- 평가기준 (평가편람)
- 작성지침 (한국어 공문 스타일)
- 고객사 콘텐츠 (자사 백데이터)

**처리**:
- LLM 기반 초안 생성 (EXAONE 3.5)
- 평가기준 섹션별 자동 배치
- 한국 공문/보고서 스타일 자동 적용

**출력**: 평가기준 준수 보고서 초안 (완벽도 80%, 사람 검수 전제)

**성공 지표**: 초안 생성 시간 1시간 내, 재작성율 < 20%

---

### 5.5 Recommendation 엔진 (Gap Analysis & Content Proposal)

**입력**:
- 현재 등급 (B)
- 목표 등급 (A)
- 등급별 벤치마크 데이터

**처리**:
- 현재 보고서 vs 상향 필요 기준 Gap 분석
- 유사 기관 A 등급 보고서 참고 콘텐츠 추출
- 실현 가능한 콘텐츠 항목 우선순위 지정

**출력**: 
- "안전 교육 이수율" → 현재 X% → 목표 Y% 상향 필요
- "중대재해 예방 투자액" → 현재 A→B 수준 콘텐츠 추가
- 예상 점수 개선: +5점 (B → A 가능성 30% → 60%)

**성공 지표**: 제안 항목 실현율 70%+, 실제 등급 개선 실적 확보

## 8. 인접 도메인 확장 로드맵

### 8.1 인접 도메인 확장 후보 (Phase 3, 우선순위 순)

#### ESG 보고서 자동화
- **시장**: 대형 공기업, 상장사 (KCGS 의무 준수 기관)
- **기술 재사용**: Document Ingestion, RAG (ESG 기준 인덱싱)
- **차별화**: 국내 ESG 기준(K-ESG) + 국제 기준(GRI/TCFD) 자동 매핑
- **기대 ARR**: +$200K

#### 감사 보고서 자동화
- **시장**: 준정부기관, 공공기관 (감사원 정기감시)
- **기술 재사용**: 초안 생성, Recommendation (감사 지적사항 예방 제안)
- **차별화**: 감사원 예시 보고서 학습 기반 사전 컴플라이언스 체크
- **기대 ARR**: +$150K

#### 면허신청서 자동화
- **시장**: 규제 대상 사업장 (의료기관, 축산시설, 식품위생 등)
- **기술 재사용**: Document Ingestion, 기준 매핑
- **차별화**: 부처별 면허기준(식약처/환경부/고용노동부) 자동 분기
- **기대 ARR**: +$100K

#### 정부 공문 자동화
- **시장**: 모든 공공기관 (보도자료, 공식 회신 등)
- **기술 재사용**: 초안 생성, 스타일 적용
- **차별화**: 중앙부처·지자체별 공문 규칙 자동 적용
- **기대 ARR**: +$200K+

#### 금융권 규제 보고서 자동화 (장기 후보, Phase 4+)
